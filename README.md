## 🛡️ Inspect before executing

PowerShell pipes downloaded scripts directly into the terminal. Anyone who prefers to inspect the installer first can use:

```powershell
irm "https://github.com/Nikolai-coder/diavlo-wav-releases/releases/latest/download/install.ps1" -OutFile "$env:TEMP\diavlo-install.ps1"

notepad "$env:TEMP\diavlo-install.ps1"

& "$env:TEMP\diavlo-install.ps1"
```

This downloads the same official installer script, opens it for inspection and only executes it afterwards.

## 🔐 Release verification

Release assets may include checksum files that allow downloaded installers to be verified.

Example:

```powershell
Get-FileHash ".\DiavloWAV-Setup-x64.exe" -Algorithm SHA256
```

Compare the generated SHA-256 value with the checksum published alongside the corresponding release.
